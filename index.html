Project Overview

DocuQuery is an advanced, privacy-first document question-answering system that transforms how you interact with your PDF documents. Powered by cutting-edge local AI/LLM technologies.

Leverages cutting-edge local AI models for secure, private document analysis with advanced AI techniques and comprehensive data protection

Uses microservices architecture for scalable and flexible document processing, enabling modular and efficient system design with independent service components

Enables semantic search and context-aware question answering with advanced natural language processing techniques and intelligent document comprehension

System Architecture & Technologies

1. Document Ingestion and Preprocessing

The system ingests PDF documents from a specified directory and prepares them for embedding and retrieval.

2. Embedding and Vector Storage

Converts document chunks into embeddings and stores them in a vector database for efficient similarity search.

3. Retrieval-Augmented Generation (RAG) Pipeline

Combines document retrieval with natural language generation to provide context-aware answers to user queries.

4. Message Handling with Kafka

Provides a streaming pipeline for handling queries and responses asynchronously in a distributed system.

5. Orchestration and Main Loop

Ties all components together into a cohesive application, orchestrated by Langchain.

6. User Interaction via GUI

A web application powered by Streamlit serves as communication platform between Client (User) and Host (RAG-based system).
